What is HTML?

HTML stands for Hypertext markup language. Markup language refers to a way of structuring and annotating text so that a computer (eg. web browser) knows how to display and interpret it.

To explain it further:

  • Mark up means you add special tags or codes around text to "mark it up"
  • These tags tell the browser what the text is (heading,paragraph,link,image,etc) and how it should appear or behave.
  • Unlike programming languages, markupmlanguages don't do computatins or logic; they just define the structure and presentation of the content.
